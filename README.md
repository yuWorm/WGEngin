### 介绍

WGEngine全称(Wap Game Engine), wap游戏引擎, 主要是为了方便简单的创建一个wap页游. 采用python + sanic实现. 并且大量使用中文变量名和方法,
降低学习成本(主要是懒的写注释和文档了,干脆直接用中文, 说实话,中文变量方法之类的还真不习惯)
